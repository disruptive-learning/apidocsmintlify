## Base URL

```
https://api.gigstack.io/v2
```

## Response Format

All responses follow a consistent format:

```json
{
  "success": true,
  "message": "Description of action",
  "data": {},
  "timestamp": 1768240724433
}
```

<Note>
  Test mode is supported throughout the API. Use `livemode: false` to work with sandbox data and test payment processor keys.
</Note>
