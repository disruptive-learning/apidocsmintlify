# Quickstart

Get up and running with the Gigstack API in 3 steps.

## 1. Get Your API Token

Go to [app.gigstack.pro/settings?tab=api](https://app.gigstack.pro/settings?tab=api) and generate a token.

## 2. List Your Clients

<CodeGroup>

```bash cURL
curl -X GET https://api.gigstack.io/v2/clients \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```javascript Node.js
const response = await fetch('https://api.gigstack.io/v2/clients', {
  headers: { 'Authorization': 'Bearer YOUR_API_TOKEN' }
});
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://api.gigstack.io/v2/clients',
    headers={'Authorization': 'Bearer YOUR_API_TOKEN'}
)
print(response.json())
```

</CodeGroup>

## 3. Create an Invoice

```bash
curl -X POST https://api.gigstack.io/v2/invoices/income \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "client": "client_abc123",
    "items": [{
      "description": "Consulting services",
      "quantity": 1,
      "price": 5000,
      "product_key": "80101500",
      "unit_key": "E48"
    }],
    "use": "G03",
    "payment_form": "03",
    "payment_method": "PUE"
  }'
```
